Requirements 
*Install Visual CADD http://tritools.com/v8_demo.php, and activate it.
*Install https://dotnet.microsoft.com/download/dotnet-core/3.1 x86 and x64 
*Set read and write permissions to all users to:
	*PTASSketchFileMigratorConsoleApp.dll.config and configurations.json file inside the root folder.
	*"xml" folder inside the root folder, create it if it does not exist.
	*C:\Documents\Visual CADD 8 folder.

*To edit application settings, edit: PTASketchFileMigratorConsoleApp.dll.config
*To edit application state, edit: configurations.json
